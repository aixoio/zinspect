# zinspect
Zip inspector

Made in Rust with Ratatui

## Usage

Run
```bash
./zinspect ./test.zip
```

Follow this format when using

```bash
./zinspect [filename]
```

Replace `test.zip` with the zip file you want to inspect

## Building/Installing

You need Rust installed to install this

### Building from source

```bash
git clone https://github.com/aixoio/zinspect.git
cd zinspect
cargo install --path .
```

and then just run `zinspect [filename]`
